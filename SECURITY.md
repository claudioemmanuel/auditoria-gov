# Security Policy — OpenWatch (Public Layer)

---

## Reporting a Vulnerability

If you discover a security vulnerability in OpenWatch:

- **Do not** open a public issue or discuss it publicly
- Email [security@openwatch.com](mailto:security@openwatch.com) with details
- We will acknowledge your report within 3 business days and coordinate a fix
- Responsible disclosure is appreciated and recognized

---

## Supported Versions

We support the latest major release of OpenWatch. Please update to the latest version before reporting vulnerabilities.

---

## Security Best Practices

- Keep dependencies up to date
- Use strong, unique passwords for all accounts
- Never share API keys or secrets
